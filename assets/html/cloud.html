Cloud Computing

Teknologi yang memungkinkan akses data dan aplikasi melalui internet kapan saja dan di mana saja.

Diagram Cloud Interaktif

Modul Cloud Computing

Infrastructure as a Service (IaaS)

Menyediakan infrastruktur digital seperti server dan penyimpanan yang dapat disewa tanpa perlu membeli perangkat keras..

Platform as a Service (PaaS)

Layanan yang memungkinkan pengembangan dan pengelolaan aplikasi langsung di cloud tanpa mengurus infrastruktur dasar..

Software as a Service (SaaS)

Aplikasi siap pakai yang dapat diakses melalui internet tanpa perlu instalasi di perangkat pengguna.

Simulasi Kode Cloud (Python)

# Upload file ke cloud (simulasi AWS S3)
import boto3

s3 = boto3.client('s3')
s3.upload_file('local_file.txt', 'my-bucket', 'cloud_file.txt')
print("File berhasil diunggah!")
